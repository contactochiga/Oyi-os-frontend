# OIS v1 Tokens

`src/app/styles/ois-tokens.css` is the Consumer copy of the Ochiga Interface System token contract.

- Tokens define semantic canvas, surface, border, text, status, spacing, radius, elevation, typography, motion, and safe-area values.
- This Phase 3B change is foundation-only: existing Consumer pages and components must not be migrated or visually redesigned here.
- Use `.ois-*` utilities only in new OIS primitives or explicitly approved component migrations.
- Consumer remains calm, residential, and awareness-first.

Phase 3C introduces shared primitives; it is the first phase allowed to migrate component styling.
