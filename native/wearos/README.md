# Oyi Wear OS Foundation Only

Wear OS is not production-ready in this repository.

The supported production companion path is currently the native SwiftUI watchOS target embedded in the Oyi Home iOS workspace.

Before Wear OS can be presented as a supported product, add:

- Android Studio Gradle project and Wear OS application module
- Kotlin + Jetpack Compose watch UI
- secure phone-to-watch session handoff
- backend watch adapter integration
- command confirmation flow
- notification channels
- emulator and real-device build verification
- Play Store signing and release assets
